# 测试集生成

构建高质量测试数据集对评估 AI 应用性能至关重要。

## 理想测试数据集的特征

- 包含高质量数据样本
- 覆盖真实世界中观察到的多种场景
- 样本数量足以得出具有统计显著性的结论
- 持续更新以应对数据漂移

人工构建这样的数据集既耗时又昂贵。Ragas 提供一套工具，用于为评估 AI 应用生成合成测试数据集。
